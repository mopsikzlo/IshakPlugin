# IshakPlugin
Этот плагин добавляет самого настоящего ишака на ваш сервер Minecraft, плагин сделан для Nukkit.
Это мой первый плагин, который сделан для Nukkit.

# Плагин тестировался на
- NukkitPetteriM1Edition
- PowerNukkitX
